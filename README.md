# SQL Sales Analysis Project

## Project Overview

This project focuses on analyzing sales data using MySQL to uncover business insights related to sales performance, profitability, customer behavior, product performance, and regional trends.

The objective of this project is to demonstrate SQL skills including data aggregation, filtering, sorting, grouping, ranking, and business analysis.

---

## Tools Used

- MySQL Workbench
- SQL
- Sample Superstore Dataset

---

## Dataset Information

The dataset contains:

- Orders
- Customers
- Products
- Categories
- Sales
- Profit
- Discounts
- Regions
- States
- Cities

Total Records: 9694

## Key Insights

- California generated the highest sales revenue.
- California and New York contributed the highest profits.
- Technology category produced strong sales and profitability.
- Some products generated losses despite high sales volume.
- The West region generated the highest profit.
- Customer profitability differed significantly from customer sales rankings.

This project helped in understanding how SQL can be used to transform raw transactional data into meaningful business insights for decision-making.
